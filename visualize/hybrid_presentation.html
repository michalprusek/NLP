Hybrid OPRO + HbBoPs

Efficient Prompt Optimization with Confidence Intervals and Early Stopping

The Problem

How do we efficiently search a large prompt space with a limited budget of LLM evaluations?

Full evaluation of each prompt candidate requires ~1300 samples - far too expensive for exhaustive search.

Our Solution

A hybrid approach combining three key techniques:

  • OPRO - Meta-LLM generates new instruction candidates
  • HbBoPs - Gaussian Process screening without LLM calls
  • Sequential Testing - Hoeffding bounds for statistically-sound early stopping

Confidence Intervals - Hoeffding Bounds

Hoeffding's Inequality:
For n independent samples with binary outcomes (correct/incorrect), with probability 1-δ:

ε = √(ln(2/δ) / (2n))

For 95% confidence (δ = 0.05): ε ≈ √(1.84/n)

Key insight: The more samples we evaluate, the narrower our confidence interval around the true accuracy. This allows us to make early decisions about prompt candidates.

Early Prompt Discard

Three Possible Decisions

DROP

upper_bound < champion

Even in the best case, this candidate cannot beat the current champion

PROMOTE

lower_bound > champion

Even in the worst case, this candidate is better than the champion

CONTINUE

intervals overlap

Need more samples to make a confident decision

GP Screening (Phase 3)

A Gaussian Process serves as a surrogate model that predicts prompt accuracy based on embeddings - without any LLM calls.

Why does this work?

  • GP learns from previous evaluations (instruction + exemplar → accuracy)
  • Deep kernel extracts features from SBERT embeddings
  • Prediction for 200 candidates takes < 1 second
  • Dramatically reduces expensive LLM calls

Conclusion

Key Contributions

  1. Hoeffding bounds enable statistically-sound early stopping decisions
  2. Multi-fidelity evaluation (10→20→40→80...) with intelligent decision-making
  3. GP screening eliminates 190/200 candidates without any LLM calls
  4. OPRO + HbBoPs combination achieves better generalization than either method alone
Result: Hybrid OPRO+HbBoPs achieves 88.93% test accuracy on GSM8K (vs 83.02% for OPRO) with the same budget of 50k evaluations.

Statistical Guarantees

The Hoeffding-based early stopping provides 95% confidence that:

  • Dropped candidates truly couldn't beat the champion
  • Promoted candidates truly are better than the champion
  • No promising candidates are incorrectly discarded
